## Installation

Clone the repository.

    git clone <repository-url>

Install dependencies.

    pip install -r requirements.txt

Create a .env file (refer to .env.example).

    API_KEY=your_api_key_here

Run the application.

    python app.py

## Dependencies

- Python 3.10+
- python-dotenv
